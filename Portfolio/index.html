Tools & Technologies

Adobe Photoshop

Adobe Photoshop is a popular image editing and manipulation software used for creating and enhancing visual content, editing and retouching photos, creating digital art and illustrations, and designing graphics and visual elements for websites and applications.

Figma

Figma is a cloud-based graphic design and user interface (UI) design tool used for creating and collaborating on visual projects, designing user interfaces and experiences (UX/UI) for websites and applications, and creating wireframes, prototypes, and high-fidelity designs.

Canva

Canva is a popular graphic design platform used for creating visual content, such as social media graphics, presentations, documents, posts, and stories.

HTML, CSS & JavaScript

These are web development tools used to design websites. HTML is a standard markup language used for creating the structure and content of web pages. CSS is a styling language used for controlling the layout, appearance, and behavior of web pages. JavaScript is a programming language used for adding interactivity and dynamic effects to web pages.
